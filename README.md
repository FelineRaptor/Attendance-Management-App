This is a terminal based app which initially asks for entering all the subject names. Then gives a list of options to either enter or view the attendance. Attendance can only be entered once in a day. Currently there is support for only a fixed number of subjects (it is set to 3 in code, if you want more subjects change the subcount variable) because having user defined subject count was breaking a lot of things. I will fix this in a later update. 
For entering attendance, first you are prompted to say if the class happened, then it asks whether you were present of absent. Currently it accepts only true or false for answer. Currently searching for View attendance by subject is case sensitive. I will fix that later. All the data is saved in a txt file named "attendanceData". 
This is the Intellij Idea Project itself. If you want to separately run just the program without an IDE, run the JavaAttendanceProj.jar file. You can run it by using the command in command prompt: java -jar "<the jar file's directory including the file's name and extension>"
